# data-analyst-retail-portfolio
Portfolio de formación en Data Analytics con enfoque en Retail Analytics. Incluye proyectos de SQL, Excel, Python, Power BI, visualización de datos, análisis de clientes y forecasting.

---
## 📌 Entregables del portafolio

### 00 · Mini Retail Database SQL

Mini base de datos relacional creada en MySQL para simular ventas de una tienda retail de productos de belleza.

El proyecto incluye creación de tablas, claves primarias y foráneas, inserción de datos, modificación de estructura con `ALTER TABLE`, actualización de registros con `UPDATE` y `CASE`, eliminación controlada con `DELETE`, consultas básicas, `INNER JOIN` y cálculo del total de venta.

**Herramientas:** MySQL · MySQL Workbench · GitHub  
**Estado:** Completado
